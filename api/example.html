YouTube Transcript API Example

How to Use the API

The YouTube Transcript API allows you to fetch transcripts from YouTube videos.

Example API URL: /api/index.py?video_id=Hc79sDi3f0U
Alternative URL format: /api/youtube/Hc79sDi3f0U

You can also specify a language (default is English):

/api/index.py?video_id=Hc79sDi3f0U&language=fr
